# Virtual Lab Setup

## Objective
To create a safe virtual environment for cybersecurity practice using Kali Linux and VirtualBox.

## Tools Used
- Oracle VirtualBox
- Kali Linux

## Steps Performed
1. Downloaded the Kali Linux VirtualBox image from the official website.
2. Imported the Kali Linux virtual machine into Oracle VirtualBox.
3. Configured memory (RAM) and processor settings.
4. Configured NAT networking for internet connectivity.
5. Started and tested the Kali Linux virtual machine.
6. Verified the setup through the Linux terminal.

## Screenshots
- VirtualBox Manager
- System Settings
- Network Settings
- Kali Linux Desktop
- Terminal Verification

## Importance of Virtual Labs
Virtual labs provide a secure and isolated environment for learning cybersecurity concepts and ethical hacking techniques without affecting the host operating system.

## Outcome
Successfully set up a Kali Linux virtual machine using Oracle VirtualBox and created a safe environment for cybersecurity learning and practice.

## Conclusion
The virtual lab was successfully created and configured. The environment is ready for practicing cybersecurity and ethical hacking activities in a safe and controlled manner.
